# Skala Basic

**Skala Basic** 프로젝트에 오신 것을 환영합니다! 이 저장소는 HTML, CSS, JavaScript 기초를 학습하기 위한 실용적인 연습 문제들을 포함하고 있습니다.

## 📚 내용

- **HTML** - 의미론적 마크업과 구조
- **CSS** - 스타일링과 레이아웃 기법
- **JavaScript** - DOM 조작과 상호작용

## 🚀 시작하기

1. 이 저장소를 클론하세요
2. 좋아하는 코드 편집기에서 파일을 열어주세요
3. 웹 브라우저에서 결과를 확인하세요

skala-basic/

## 💡 학습 목표

- 반응형 웹 페이지 구축

- 상호작용하는 JavaScript 코드 작성
- 모범 사례와 깔끔한 코드 연습
